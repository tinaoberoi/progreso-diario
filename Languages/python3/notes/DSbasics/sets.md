# Sets in Python3

* Sets are an unordered collection of unique elements.
```python
>>x = set()
>>x.add(1)
>>x
{1}
```
```python
# Create a list with repeats
>>list1 = [1,1,2,2,3,4,5,6,1,1]

# Cast as set to get unique values
>>set(list1)

{1, 2, 3, 4, 5, 6}
```
